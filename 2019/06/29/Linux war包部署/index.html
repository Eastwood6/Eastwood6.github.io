Linux war包部署

ssh工具 Xshell

如果还没配置JDK参照 linux查看java jdk安装路径和设置环境变量

  1. 项目打成war包
  2. xshell 通过fstp传输至Linux
  3. 将war包移至tomcat的webapp目录
  4. ./startup.sh后tomcat会自动加载war包成文件夹
  5. tail -f logs/catalina.out查看tomcat运行状态(需要返回上一层目录)
  6. 至此大致完成。
