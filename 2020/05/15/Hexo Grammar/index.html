安装

  • nodejs,cnpm:

首先要安装 Node.Js,顺带会有npm

本来是用npm来安装Hexo,但因为国内镜像系统较慢，利用npm安装cnpm也就是淘宝

npm install -g cnpm --registry=https://registry.npm.taobao.org

node -v验证 npm -v验证

  • hexo:

安装Hexo:cnpm install -g hexo-cli

hexo -v验证

初始化

  • 初始化博客

mkdir blog

cd blog

sudo hexo init

  • 启动博客

hexo s用来预览

  • 创建博客

hexo n "name"

部署远端(cg:github)

安装git部署插件： cnpm install --save hexo-deployer-git

配置远端地址_config.yml:

vim _config.yml
